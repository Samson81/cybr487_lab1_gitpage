Getting Started

Setup and Running

1 - Cloning the Repository

  1. Make sure git and python 3 are installed on your computer

  2. In a new terminal, run the command:

    • git clone https://github.com/Samson81/cybr487_lab1.git

  3. Run the command to move into the repository’s directory:

    • cd cybr487_lab1

2 - Setting up Virtual Environment and Requirements

Run the following in the same terminal/directory:

  1. python3 -m venv venv

  2. source venv/bin/activate

  3. pip install -r requirements.txt

3 - Running Network Analyzer

Run the following in the cyber487_lab1 directory

  1. sudo python3 analyzer.py -i <ip_address>
