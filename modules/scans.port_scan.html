scans.port_scan.port_identify module

Identification of services running on a port

scans.port_scan.port_identify.identify(data: bytes) → str | None[source]

Attempts to identify a service running on a port based on what it returns.

Services that can be detected: - SSH (Any that follow RFC 4253 and those that don’t) - FTP (vsFTPd, FileZilla and others) - HTTP - Telnet (Only inetutils version) - MySQL (Most likely only MariaDB) - SMTP

Parameters:

data (bytes) – The data returned from the socket connection

Returns:

The identifying information of the service, or None if no regex exists for the service

Return type:

(str | None)

scans.port_scan.service_scan module

The module for running service scans

scans.port_scan.service_scan.service_scan(ip: dict[str, Address_Info], timeout: int | float = 5) → None[source]

Attempts to identify the actual service running on a port. Adds the data to a dictionary in the Address_Info object

Services that can be detected: - SSH (Any that follow RFC 4253 and those that don’t) - FTP (vsFTPd, FileZilla and others) - HTTP - Telnet (Only inetutils version) - MySQL (Most likely only MariaDB) - SMTP

Parameters:

  • ip (dict[str, Address_Info]) – The dictionary of Address_Info objects to scan

  • timeout (int | float = 5) – The timeout for each socket
