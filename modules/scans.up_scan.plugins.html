scans.up_scan.plugins.arp module

Function for scanning IP ranges with ARP packets

scans.up_scan.plugins.arp.scan(ip: str | list[str], timeout: int = 2) → list[str][source]

Performs an ARP packet scan on the provided IP range.

Sends an ARP “Who is at x.x.x.x? Tell y.y.y.y” packet to each IP address.

Parameters:

  • ip (str | list[str]) – A string or list that contain the IP addresses to check

  • timeout (int) – The time in seconds to wait for a packet

Returns:

A list of the IP addresses that responded

Return type:

str

scans.up_scan.plugins.nmap module

Function that mimics the default nmap scan

scans.up_scan.plugins.nmap.scan(ip: str | list[str], timeout: int = 2) → list[str][source]

A scan that mimics the default nmap scan.

This scan sends the following packets to each IP address: - ARP - ICMP Echo - ICMP Timestamp - TCP ACK on port 80 - TCP SYN on port 443

Parameters:

  • ip (str | list[str]) – A string or list that contain the IP addresses to scan

  • timeout (int) – The time in seconds to wait for a packet. Default 2

Returns:

A list that contains all the IP addresses from the 5 scans

Return type:

list[str]

scans.up_scan.plugins.tcp module

Function for scanning IP ranges with ICMP packets

scans.up_scan.plugins.tcp.scan(ip: str | list[str], timeout: int = 2, port: int = 80, flags: str = 'S') → list[str][source]

Performs a TCP packet scan on the provided IP range.

Sends a TCP packet with the following fields:

  • port: 80 (or the chosen port)

  • flags: SYN (or the chosen flag)

Parameters:

  • ip (str | list[str]) – A string or list that contain the IP addresses to check

  • timeout (int) – The time in seconds to wait for a packet

  • port (int) – The destination port to send the TCP packet to

  • flags (str) – The TCP flags to set in the packet

Returns:

A list of the IP addresses that responded

Return type:

list[str]
