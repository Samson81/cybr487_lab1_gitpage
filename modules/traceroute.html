traceroute module

Traceroute functionality using ICMP or TCP packets

traceroute.icmp_traceroute(target_ip: str, timeout: int, max_hops: int) → list[str][source]

Perform ICMP-based traceroute

Uses ICMP Echo Request packets with incrementing TTL values. Each hop is probed 3 times to handle packet loss.

Parameters:

  • target_ip (str) – Destination IP address

  • timeout (int) – Timeout for each probe

  • max_hops (int) – Maximum hops to try

Returns:

Formatted hop information

Return type:

list[str]

Probe a single hop using ICMP, attempting 3 times

  • ttl (int) – Time-to-live value for this hop

  • attempts (int) – Number of attempts per hop (default 3)

Returns:
